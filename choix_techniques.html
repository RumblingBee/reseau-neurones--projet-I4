Accueil (current) Choix techniques Fonctionnement

Nous avons choisi d'utiliser l'API tensorflow car c'est une API très utilisée et bien documentée qui permet de créer des réseaux de neuronnes simplement.
De plus, utiliser tensorflow était l'occasion de tester une API utilisée dans le monde professionnel.
Enfin, bien que moins performante que la version Python, nous avons choisi d'utiliser la version Javascript car elle est bien plus facile à configurer.
